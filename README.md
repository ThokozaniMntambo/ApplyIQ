# ApplyIQ 🚀

ApplyIQ is a modern job application tracking platform that helps users organize and manage their job search efficiently.

Users can create an account, track applications, monitor interview progress, and manage job statuses all in one clean dashboard.

---

## ✨ Features

- 🔐 User Authentication (Register/Login)
- 📊 Dashboard with application statistics
- ➕ Add new job applications
- ✏️ Update application status
- 🗑️ Delete applications
- 📂 Filter applications by status
- 📱 Responsive and clean UI
- 🔒 Protected routes and user-specific data
